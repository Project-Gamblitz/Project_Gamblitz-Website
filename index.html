What is "Project_Gamblitz"?

"Project_Gamblitz", also known as the Splatoon 1 plaza mod, is a modpack for Splatoon 2 which adds new things from the original Wii U game, such as special weapons, clothing, extra features and more.

Check the full list of features here.

How do I install the mod?

First of all, make sure you have installed Atmosphere in your Nintendo Switch by following this guide, it's highly recommended to create an emuNAND.

Then, visit the GameBanana Page for the download and the Installation PDF for the instructions.

Alternatively, you can use the following installation tutorial video.

Can I install the mod without a PC?

NOt yet, but a Homebrew App is planned to install the mod directly from the Nintendo Switch if you already have Custom Firmware.

Does the mod work on emulators?

Yes! But only with a limited amount of features. Also note that we just support Ryujinx and not Yuzu.

Does the mod support local multiplayer?

Yes, both of LAN and Local Play multiplayers modes work. However, every console must be running the mod.

Can I stream/make videos about the mod?

You can make videos, take screenshots, etc; as long you credit the team.
